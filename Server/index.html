Welcome to FOTA

Automatic configuration update tool.

What's FOTA ?

The Project is mainly targeting to upgrade the firmware of remote Hardware ECUs OTA (Over the air).
The project is an initalization phase to create a portable, configurable platform to update remotely ECUs with different methods and Hardware.

Select HEX file to upload

How it works

A new software version is

uploaded to the server.

The main ECU receive

the file using WIFI

The Application ECU receive

the file using UART and flash it.
